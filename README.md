# MiniEngine

A lightweight 2D game engine written in Python using Pygame.

## Goal
The goal of this project is to build a simple but extensible game engine from scratch to understand how real engines work internally.

## ️ Features (in progress)
- Game loop
- Scene system
- Game objects
- Sprite rendering
- Input handling
- Collision system (planned)

## Tech Stack
- Python 3
- Pygame

## Status
Work in progress (learning project)

## Structure
engine/ – core engine code  
examples/ – sample games  
assets/ – images & sounds  

## Why this project?
Built for learning game engine architecture, OOP design, and game loops.
